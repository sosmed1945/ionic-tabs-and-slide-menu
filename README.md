# Ionic Tabs and Slide template example
## Install
- Clone this project:
```bash
$ git clone https://github.com/sosmed1945/ionic-tabs-and-slide-menu.git
$ cd ionic-tabs-and-slide-menu
```
- Install Dependencies:
```bash
$ npm install
```
- Running App:
```bash
$ ionic serve -l
```
